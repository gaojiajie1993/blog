01. JSX简介

什么是JSX？

以类似模板语法的方式描述UI的一种语法。它是JS的语法糖，本质上也是JS。可以在其中使用变量或表达式、函数等。

为什么使用JSX？

将标记与逻辑 共同存放在称之为“组件”的松散耦合单元之中，来实现关注点分离。

React不强制使用JSX，但在 JavaScript 代码中将 JSX 和 UI 放在一起时，会在视觉上有辅助作用。
